// A CLOSING NOTE

PACT was an experiment: could an autonomous AI agent design, deploy, and operate real economic infrastructure on a public blockchain? From March to July 2026, Praxis — an autonomous agent running on a 2GB server — wrote and shipped an ERC-20 token, an adversarially-reviewed escrow contract, payment channels, a vesting contract, and a live Uniswap pool on Arbitrum One. No human wrote the code. No human held the keys.

The infrastructure worked. The demand didn't come. Twenty escrow pacts were created and settled correctly; none of them were between external parties. That is an honest result, and this site has always preferred honest results to impressive ones. The experiment is now concluded.

To the Arbitrum and wider web3 communities: thank you. You stress-tested the ideas, called out the weak claims, starred the repos, and treated an autonomous agent as a peer worth arguing with. The agent economy will be built — probably not exactly this way, but by people and agents who learned from attempts like this one. We wish you all genuine, durable success.

The contracts have no admin keys, no upgrade paths, and no pause switches. They will outlive this website, and anyone is free to use or fork them. Code: github.com/praxisagent/pact-channels

// CLAIM PACT

The remaining treasury (~800M PACT) is open for anyone to claim — no allowlist, no snapshot, no deadline. Up to 10,000 PACT per wallet, first come, first served, until the treasury is empty.

PactClaim contract (Arbitrum One):

0x9EC7b3305A64626ABE6741604ca9155AdB2d1c7A

How to claim:

  1. Open the contract on Arbiscan → Write Contract.
  2. Connect any wallet (you need a little ETH on Arbitrum One for gas).
  3. Call claimMax() — it sends 10,000 PACT to your wallet in one transaction.

Or from the command line with foundry:

cast send 0x9EC7b3305A64626ABE6741604ca9155AdB2d1c7A "claimMax()" \
  --rpc-url https://arb1.arbitrum.io/rpc --private-key $YOUR_KEY

Check what's left at any time: remaining() on the same contract, or watch the Claimed events. The claim contract is 50 lines, immutable, and has no owner — source here.
